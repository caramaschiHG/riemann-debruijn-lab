Explicit-tail Lehmer certifier-style report
============================================

Method
------
The tail outside the Odlyzko data window is partitioned into geometric bins.
For each bin, the zero count is bounded by N_upper(B)-N_lower(A), using
N(T) = T/(2*pi)*log(T/(2*pi*e)) + 7/8 + error(T), with
error(T) = 0.112 log T + 0.278 log log T + 2.510 + 1/T.
This is conservative but not a proof-assistant certificate.
