# Security Policy

This is a computational research repository. It should not process untrusted
files with elevated privileges.

## Reporting

For security issues, open a private security advisory if hosted on GitHub, or
contact the maintainers through the repository's configured contact channel.

## Scope

Security reports may include:

- unsafe parsing of external data,
- path traversal in loaders or scripts,
- command injection in scripts,
- dependency risks affecting reproducibility.

Mathematical correctness issues should use the mathematical claim review issue
template instead.
